# 二手交易管理系统 java124

## 项目概述

二手交易管理系统是一个基于B/S架构和MVC设计模式的web网站。本项目采用前后端分离的开发模式，前端使用Vue框架，后端采用Java语言和SpringBoot框架。

## 技术栈

- **后端**：Java, SpringBoot, Mybatis
- **前端**：Vue
- **数据库**：Mysql
- **开发工具**：IDEA, JDK1.8, Maven

## 系统角色

- **用户**：可以进行物品发布、查询、购买及个人中心管理等功能
- **管理员**：负责交易物品、订单信息及用户信息的管理

## 功能模块

### 用户功能

1. **首页**
   - 全部查询
   - 分类查询
   - 关键字搜索
2. 发布物品
3. 查看消息
4. **个人中心**
   - 编辑个人信息
   - 编辑收货地址
5. **状态查询**
   - 我发布的
   - 我下架的
   - 我收藏的
   - 我卖出的
   - 我买到的

### 管理员功能

1. **交易物品管理**
   - 上线的二手物品（违规下架）
   - 下架的二手物品（永久删除）
2. **订单信息管理**
   - 删除订单
3. **用户信息管理**
   - 正常用户（封号）
   - 违规用户（解封）
   - 管理员（添加管理员）
